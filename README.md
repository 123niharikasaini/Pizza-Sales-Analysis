# Pizza Sales Analysis

### Project Overview

This project presents a Pizza Sales Analysis Dashboard aimed at uncovering key business insights from sales data. The analysis focuses on understanding customer ordering behavior, sales performance, and product trends to support better business decisions.

Special thanks to the Data Tutorials YouTube channel for providing the dataset and inspiration for this project.

### Purpose

The objective of this project is to analyze pizza sales data and visualize key performance indicators to help stakeholders:

1. Understand sales patterns and customer behavior
2. Identify top-performing and underperforming products
3. Optimize sales strategies and operational planning
   
### Tools & Technologies

1. SQL
2. Microsoft Excel
3. Data Cleaning & Transformation
4. Data Analysis & Visualization
5. Data Source

The dataset is sourced from the Data Tutorials YouTube channel and contains transactional-level pizza sales data.

##### Key fields include:

Order details: order_id, order_date, order_time
Product details: pizza_name, category, size, ingredients
Sales metrics: quantity, unit_price, total_price

### Key Performance Indicators (KPIs)

1. Total Revenue - Total income generated from pizza sales
2. Average Order Value - Average revenue per order
3. Total Pizzas Sold - Total quantity of pizzas sold
4. Total Orders - Total number of orders placed
5. Average Pizzas per Order - Average pizza quantity per order
   
### Dashboard Components

1. Sales Trends Analysis
1. Bar chart for daily order trends
2. Line chart for hourly order patterns
2. Category & Size Insights
1. Donut chart showing % of sales by pizza category
2. Pie chart showing % of sales by pizza size
3. Funnel chart representing pizzas sold by category
3. Product Performance
1. Top 5 best-selling pizzas (horizontal bar chart)
2. Bottom 5 least-selling pizzas (horizontal bar chart)

### Key Insights
1. Peak order days are Thursday, Friday, and Saturday, indicating higher demand toward weekends
2. Highest order volume occurs during 12-1 PM and 4-8 PM, aligning with lunch and dinner hours
3. Classic pizza category contributes the highest to both revenue and total orders
4. Large-sized pizzas generate the maximum share of sales
5. Top performers: The Classic Deluxe Pizza and Barbecue Chicken Pizza lead in sales
6. Low performer: The Brie Carre Pizza records the lowest sales
